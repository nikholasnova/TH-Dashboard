# Image Evidence Checklist

Use this folder for project screenshots/photos.

## Required Files

- `node1-circuit.jpg` - Node 1 wiring and hardware close-up
- `node2-circuit.jpg` - Node 2 wiring and hardware close-up
- `measurement-setting.jpg` - Physical placement of one or both nodes
- `dashboard-two-nodes.png` - Web dashboard showing both `node1` and `node2`
- `supabase-readings-proof.png` - Supabase table showing rows from both node IDs

## Caption Template

For each image, include 1-2 lines in the main `README.md`:

- what is shown,
- why it verifies system behavior (wiring correctness, deployment setting, two-node data, etc.).
